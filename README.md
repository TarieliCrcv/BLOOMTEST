This project (BLOOMTEST) is a complete (automation, manual, api) testing of "https://bloomaroma.ge". the main purpose is fixing bugs, errors and identifing problems to improve overall performance and marketing of the brand as well as adding project to my porfolio.

Automation tests are written with Playwright (typescript language). API tests are done using postman.

Agile/Scrum methodology used to simplify management of testing and development and get experience.

Test cases, documentations and etc. can be viewed in "docs" file

to run automated tests type 'npx playwright test'
